# ReactSSR
企业门户网站前端

## 概要和目的
- 做一个企业的门户网站，先做前端页面，用React的服务端渲染。
目的是要学习React的服务端渲染知识。
- 如果可能的话，同样做一套预渲染的页面，比较他们之间的优缺点和差异。
